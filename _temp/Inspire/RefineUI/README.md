# RefineUI

RefineUI is a minimalist World of Warcraft UI addon for **Retail** that refines the default Blizzard UI instead of replacing it.

## Philosophy

**Refine, don't replace.**

RefineUI focuses on improving the look, layout, and usability of Blizzard's existing frames while keeping the core UI familiar. By utilizing and building off of the default Blizzard frames rather than creating its own, it remains incredibly lightweight and highly performance-focused.

This isn't a UI with a million options and endless customizations—it is designed to be simple, functional, and work right out of the box for all types of players.

Ultimately, RefineUI is built for me and my own personal taste and aesthetic preferences, and I freely share it for anyone else who enjoys a clean, intuitive, what-you-see-is-what-you-get experience.

## What It Includes

- Default UI styling and skinning
- Unit frame improvements
- Nameplate improvements
- Action bar refinements
- Chat and tooltip polish
- Bags, maps, quest, and other quality-of-life modules
- Native Edit Mode integration support
- And much more!

## Configuration

As previously mentioned, there isn't a gigantic options panel to overwhelm you. All key addon settings are designed to be intuitive and accessible exactly where you expect them:

- **Edit Mode:** Fully integrated with Blizzard's native Edit Mode for layout, positioning, and configuring custom additions.
- **Frame Options:** Additional settings are editable through in-game menus attached directly to their relevant frames.

## Install

RefineUI is packaged from the repository root, not from the `RefineUI` addon folder by itself.

### Manual Install

1. Download this repository as a ZIP, then extract it.
2. Open the extracted `RefineUI` folder.
3. Copy the top-level `Fonts` and `Interface` folders into your WoW Retail folder:
   `World of Warcraft\\_retail_\\`
4. When prompted, allow Windows to merge folders.
5. Restart WoW or run `/reload`.

This should leave the addon at:
`World of Warcraft\\_retail_\\Interface\\AddOns\\RefineUI`

### Git Install

If you are cloning the repo directly, you can keep the repository anywhere on disk.

Copy or sync the repo's `Fonts` and `Interface` folders into:
`World of Warcraft\\_retail_\\`

Do not install from `Interface\\AddOns\\RefineUI` alone, because the repository also includes shared assets outside the addon folder.
